Задание:
В одномерном массиве uint32_t, содержащем столбцы матрицы, провести преобразования:
Назовем характеристикой строки результат поразрядной операции "штрих Шеффера" между элементами строки. 
Осуществить циклический сдвиг каждой строки на количество единиц в пяти старших битах значения характеристики.
